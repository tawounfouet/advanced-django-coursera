## View Blog
An endpoint at /api/v1/whoami/ (the WhoAmIView class) has been set up to return the username of the current user, in a format like this:
{"username": "<username>"}

Or if no user is logged in, the value will be null:
{"username": "null"}

You can use this URL to test that you’ve implemented the authentication correctly. You don’t need to make any changes to this view.
